# SmartData: Automated Data Cleaning & EDA Tool 📊

An automated pipeline to transform messy datasets into analysis-ready data using **Pandas** and **NumPy**.

### 🛠️ Key Functionalities:
- **Missing Value Imputation:** Smartly fills nulls using median/mean logic.
- **Outlier Detection:** Identifies anomalies in numerical columns.
- **Data Standardization:** Normalizes column names and formats for ETL pipelines.
- **EDA Visualization:** Basic plotting of data distributions using Matplotlib.

### 🚀 Tech Stack:
- **Primary Library:** Pandas (for Data Transformation)
- **Visualization:** Seaborn & Matplotlib
- **Core:** Python & NumPy
